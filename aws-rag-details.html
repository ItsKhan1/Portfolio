AWS Rag Chatbot

A LangChain-Powered AWS RAG System for Automotive Customer Support with Dual Knowledge Bases

Tired of resolving user queries manually while juggling business tasks? Try SAMACO’s smart chatbot – built with AWS to assist with inventory and general car-related queries instantly.

Built with state-of-the-art LLMs and optimized for complex dealership operations, this intelligent system converts scattered data and documents into real-time, conversational support. Featuring multi-agent architecture and vector search, it provides fast, context-aware responses to boost decision-making. Tailored for inventory, vehicle specs, and customer queries, the chatbot integrates AWS and LangChain tools to deliver scalable, domain-specific automation. The result: smarter operations, quicker resolutions, and a premium digital experience for SAMACO customers.

Highlights


  Developed a LangChain + Claude Sonnet RAG pipeline to automate responses for inventory checks, car specs, and general inquiries.


  Integrated Bedrock Knowledge Bases as a vector store for fast, accurate document retrieval using embeddings tailored to SAMACO data.


  Added custom query rewriting logic to improve user input clarity and enhance semantic search performance in the automotive context.


  Built a robust FastAPI backend with React UI and WebSocket support for real-time, chat-like user interaction with the SAMACO assistant.

Key Features

Friendly User Interface

SAMACO’s AI assistant puts a dynamic and user-friendly interface at the center of the experience, enabling seamless interaction with intelligent automotive data. Built with a modern React-based UI, it supports threaded chats and persistent history, allowing users to pick up conversations where they left off. The interface is highly customizable — allowing control over LLM model version, temperature, memory size, and response structure, making it ideal for testing and adapting different setups. With intuitive onboarding, sample queries, and advanced filtering, both technical and non-technical users can navigate and utilize the system efficiently. The result: a flexible, enterprise-grade chatbot tailored for real-world automotive workflows.
