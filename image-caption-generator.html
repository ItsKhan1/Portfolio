Image Captioning

Automatic Image Captioning with Flickr8k and Deep Learning

It bridges the gap between vision and language, making visual content more accessible and searchable.

This project focuses on building an AI model that generates human-like captions for images using the Flickr8k Dataset. By combining InceptionV3 for image feature extraction and LSTM networks with GloVe embeddings for caption generation, the system learns to understand and describe visual scenes automatically. The primary goal is to improve accessibility for visually impaired users and enhance image indexing in digital platforms.

Used InceptionV3

Extracted rich visual features from each image for deeper scene understanding.

LSTM Decoder

Generated grammatically coherent captions aligned with visual input.

GloVe Embeddings

Improved semantic consistency and generalization of the generated text.

BLEU Scoring

Trained and evaluated on Flickr8k using BLEU scores for performance validation.

Friendly User Interface

The interface is designed with a clean, intuitive layout to make the captioning process simple and engaging. Users can upload any image and receive an automatically generated caption within seconds. Built with React, the frontend supports live previews, image history, and downloadable captions. The backend, powered by Flask or FastAPI, handles model inference and manages interactions efficiently.
