Agentic Workflow

LangGraph RAG-Based Multi-Agent Workflow for Pharmacy Customer Support

Tired of resolving user queries manually very hectic along managing the business?

Stay ahead of the innovation curve with the Chemetall BASF AI Chatbot, a cutting-edge RAG-powered solution built for intelligent enterprise assistance. Designed with advanced LLMs and optimized for complex industrial data, this powerful system transforms document-based insights into real-time conversational experiences. With the BASF chatbot, your organization benefits from seamless integration of multi-agent frameworks and vector databases, enabling context-aware responses that accelerate decision-making. Specifically tailored for chemical industry workflows, this state-of-the-art web application extracts, processes, and interprets technical documents from diverse sources using AWS and Azure intelligence tools. The result is an AI-driven assistant that delivers reliable, scalable, and domain-specific support to elevate BASF's operational efficiency and customer engagement.

Highlights


  Implemented LangGraph-powered multi-agent workflows with GPT-4o, enabling coordinated task handling, intelligent decision flows, and chat history management.


  Utilized Azure Cosmos DB as a vector store, leveraging OpenAI’s large text embeddings to enable efficient and accurate document retrieval.


  Created an n-gram based next-word predictor to intelligently rephrase user queries before vector search, boosting response relevance.


  Built a scalable FastAPI backend with a React-based frontend and integrated WebSocket support for real-time response streaming.

Key Features

Friendly User Interface

Chemetall BASF’s AI assistant prioritizes a seamless and intuitive user experience to maximize the value of intelligent document interaction. The modern React-based web application is thoughtfully designed with threaded conversations and persistent chat history, allowing users to revisit previous interactions effortlessly. Advanced filtering options enable users to refine queries and retrieve precise insights quickly. For new users, the interface provides clear onboarding guidelines, including sample queries and feature walkthroughs, making it easy to get started. This clean and functional design ensures both technical and non-technical users can engage confidently, transforming complex document analysis into a smooth conversational experience.
