About

Hey! I’m a passionate Generative AI and NLP enthusiast with hands-on experience in building real-world AI systems. I’ve worked on a wide range of projects including classification, topic modeling, heading prediction, and retrieval-augmented generation (RAG) multi-agents workflow using transformer models and LLMs like ChatGPT. I’ve processed over 800 documents using AWS Textract and Azure Document Intelligence to extract high-quality data for modeling. My experience also includes designing RAG pipelines with vector search using PGVector and Cosmos DB, as well as building scalable backend services with FastAPI, PostgreSQL, and cloud storage like AWS S3 and Azure Blob.

I’ve engineered multi-agent systems using LangChain, LangGraph, and Crew AI, deploying them on AWS Bedrock and Lambda for intelligent orchestration. Additionally, I’ve worked on real-time object detection models using TensorFlow and developed academic projects like image captioning with CNN-RNN and sentiment analysis using transformer pipelines. I'm always exploring new ways to push the boundaries of AI and love working on systems that solve meaningful problems.
